# <a name="aspnet-core-docs"></a>ASP.NET Core docs

Toto úložiště obsahuje dokumentaci koncepčního ASP.NET Core, která je hostována na [docs.Microsoft.com/ASPNET/Core](https://docs.microsoft.com/aspnet/core). Pokud chcete mít nápovědu, přečtěte si [příručku pro přispívajícího](CONTRIBUTING.md) a [seznam problémů](https://github.com/dotnet/AspNetCore.Docs/issues) .

Změny v dokumentaci k rozhraní API se provádí v [úložišti AspNetApiDocs](https://github.com/dotnet/AspNetApiDocs) proti trojnásobku lomítka `///` komentářů.

ASP.NET 4. x – změny dokumentace jsou provedeny v [úložišti dotnet/AspNetDocs](https://github.com/dotnet/AspNetDocs).
