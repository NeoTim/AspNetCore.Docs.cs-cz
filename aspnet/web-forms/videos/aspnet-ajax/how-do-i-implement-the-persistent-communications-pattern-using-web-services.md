<a name="how-do-i-implement-the-persistent-communications-pattern-using-web-services"></a>[Jak na:] Implementace vzoru trvalé komunikaci pomocí webových služeb?
====================
podle [Stagner Jan](https://github.com/JoeStagner)

Tradiční webu v prohlížeči a serverem neudržují nepřetržitě komunikují, ale komunikovat pouze v reakci na provádění akce uživatele. Moderní webu kde stránce stane kontejnerem aplikace může být výhodné pro prohlížeče a serveru pro uchování nepřetržitě komunikují, tak, aby se aktualizace stránky může dojít, aniž by uživatel provádění akce. To se označuje jako trvalé vzor komunikace pro AJAX. Technologie ASP.NET AJAX nabízí dva způsoby hlavní vývojářům webů implementovat trvalé vzor komunikace. V dřívějších videu jsme viděli, jak používat technologie ASP.NET AJAX UpdatePanel jako základ pro implementaci. V tomto videu jsme zjistěte, jak implementovat pomocí JavaScrpt volání webové služby, který odebírá potřebu, ovládacího prvku ASP.NET AJAX UpdatePanel stejného vzoru.
