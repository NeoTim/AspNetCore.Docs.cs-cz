<a name="how-do-i-use-the-aspnet-ajax-updateprogress-control"></a>[Jak na:] Použití ovládacího prvku ASP.NET AJAX UpdateProgress?
====================
podle [Stagner Jan](https://github.com/JoeStagner)

V aplikaci AJAX je často znatelné pozastavení mezi době, kdy prohlížeč odešle požadavek sítě a časem v prohlížeči obdrží odpověď. Aby mohl uživatel vědět, že síťovou konverzaci probíhající, UpdateProgress AJAX technologie ASP.NET poskytuje ovládací prvek okamžitou zpětnou vazbu zobrazením animovaný bitové kopie, 'načítání' text nebo jiné HTML, které zmizí po obdržení odpovědi sítě.
