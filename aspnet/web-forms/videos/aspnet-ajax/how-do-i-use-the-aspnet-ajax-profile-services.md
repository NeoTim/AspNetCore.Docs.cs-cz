<a name="how-do-i-use-the-aspnet-ajax-profile-services"></a>[Jak na:] Použít profil služby ASP.NET AJAX?
====================
podle [PEL Jan](https://twitter.com/chrispels)

ASP.NET 2.0 zavedená [profilu vlastnosti](https://msdn.microsoft.com/library/at64shx3.aspx) pomocí kterého můžete uložit podrobnosti pro každého uživatele vaší webové aplikace ASP.NET. ASP.NET AJAX profilu služby umožňují vaší klientské webové rozhraní pro načtení vlastnosti profilu uživatele z webové aplikace na straně serveru. Toto video ukazuje, jak ukázkový server technologie AJAX cesta může ukládat informace o žádosti odeslané cesta uživatele tak, aby při dalším návštěvníky webu se dá načíst podrobnosti.
