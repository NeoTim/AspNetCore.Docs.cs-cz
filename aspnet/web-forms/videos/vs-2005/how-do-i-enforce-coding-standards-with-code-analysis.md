<a name="how-do-i-enforce-coding-standards-with-code-analysis"></a>[Jak na:] Vynutit kódování standardy s analýzou kódu?
====================
podle [Menegay Jan](https://twitter.com/CMenegay)

Pomocí sady Visual Studio Team System máte dobrou kontrolu pravidel analýzy kódu, která se týkají projektů knihovny tříd, webové projekty a správy zdrojového kódu. V tomto videu jsme informace o zadání chyby a upozornění, která se použije při projektu sestavení a kód zkontrolujte in a nám zjistit, jak potlačit konkrétní pravidlo z použití členem určité třídy. Pomocí této možnosti konfigurace:, jejichž dodržování může získat kódování standardy ke zlepšení kvality kódu, a zajistit nepoužitelných pravidla nikdy vaší způsobem.
