<a name="how-do-i-implement-continuous-integration-with-team-foundation"></a>[Jak na:] Implementovat průběžnou integraci s Team Foundation?
====================
podle [Menegay Jan](https://twitter.com/CMenegay)

Existuje řada průběžnou integraci nástrojů, které pracuje na Team Foundation Server a v tomto videu se podíváme na bezplatnou [Norton TeamCI](http://teamsystemrocks.com/files/12/tools/entry1018.aspx) nástroje, které jsou k dispozici z [Team System skály](http://teamsystemrocks.com/). TeamCI pravidelně kontroluje, zda aktualizovat soubory se změnami pro Team Foundation Server a pokud ano, sestavte řešení a spuštění všech automatizovaných testů.
