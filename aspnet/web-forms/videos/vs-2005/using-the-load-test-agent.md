<a name="using-the-load-test-agent"></a>Pomocí agenta Test zatížení
====================
podle Erica Lee

V tomto videu jsme zkontrolujte jak začlenit zatížení testovacího agenta v testování, které jsme dělat s Visual Studio Team System. Načíst agentem Test Agent je součástí "zařízení", může být nasazena na jednotlivé počítače, které chcete povolit zatížení, přízvuk a z jiného operačního systému nebo počítač konfigurace testování částí. Každé zařízení můžete pracovat s několika agenty a mohou být nasazeny na virtuální nebo vzdáleného počítače, povolení pro komplexní testování pomocí sady Visual Studio Team System.
