<a name="how-do-i-create-a-custom-provider-for-logging-health-monitoring-events"></a>[Jak na:] Vytvoření vlastního zprostředkovatele pro protokolování událostí monitorování stavu
====================
podle [PEL Jan](https://twitter.com/chrispels)

Jan PEL ukazuje postup vytvoření vlastního zprostředkovatele stavu ASP.NET monitorování systému, který zaznamenává události do souboru založený na textu protokolu sledování stavu. Naučte se vytvořit třídu vlastního zprostředkovatele, která dědí z třídy BufferedWebEventProvider, pochopit metody a vlastnosti, které slouží k vytvoření zprostředkovatele a vytvořit web, který používá k zápisu souborů protokolu.
