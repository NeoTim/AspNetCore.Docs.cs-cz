<a name="how-do-i-implement-a-custom-health-monitoring-event"></a>[Jak na:] Implementovat vlastní stavu monitorování událostí
====================
podle [PEL Jan](https://twitter.com/chrispels)

V této video PEL Jan zobrazí jak vytvořit vlastní stavu monitorování událost, která rozšiřuje standardní stavu monitorování událostí technologie ASP.NET. Vlastní poskytuje informace o další události a zformátuje ho pro sledování událostí zprostředkovatele stavu. Vlastní modul HTTP je poté jste vytvořili pro vyvolání vlastní stav monitorování událostí explicitně, protože jenom zvyšuje standardní stav monitorování událostí rozhraní .NET Framework. Kromě toho je vlastní stav monitorování událost vyvolána explicitně na stránce za kód pro stránky .aspx.
