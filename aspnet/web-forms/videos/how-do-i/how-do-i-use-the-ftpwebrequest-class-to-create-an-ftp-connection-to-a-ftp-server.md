<a name="how-do-i-use-the-ftpwebrequest-class-to-create-an-ftp-connection-to-a-ftp-server"></a>[Jak na:] Použití třídy FtpWebRequest k vytvoření připojení k serveru FTP na serveru FTP
====================
podle [PEL Jan](https://twitter.com/chrispels)

V této video PEL Jan ukazuje, jak používat třídu FtpWebRequest vytvořit připojení FTP a přenos souboru. Nejprve vytvořte formuláři jednoduché .aspx a vyberte soubor, který chcete převést na server FTP. V dalším kroku najdete v části základní metody a vlastnosti třídy FtpWebRequest, která je nutné nakonfigurovat, aby bylo možné navázat připojení k serveru FTP. Potom najdete v části Jak získat obsah místního souboru, soubor lze převést pomocí třídy StreamReader a třídy StreamWriter k zápisu do FtpWebRequest. Nakonec se získávají FtpWebResponse určit výsledek přenos. Tento přístup umožňuje prostřednictvím kódu programu přenos souborů pro webovou aplikaci.
