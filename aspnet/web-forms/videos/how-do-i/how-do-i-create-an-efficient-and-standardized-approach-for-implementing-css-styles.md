<a name="how-do-i-create-an-efficient-and-standardized-approach-for-implementing-css-styles"></a>Jak I: vytvořit efektivní a standardizovaném přístup pro implementaci stylů CSS
====================
podle [PEL Jan](https://twitter.com/chrispels)

V této video PEL Jan ukazuje, jak uspořádat a implementovat stylů CSS, které poskytují standardizované vzhled a chování na web. Kromě toho jsou uspořádány stylů zajistit maximální flexibilitu a – opětovné použití. Sady stylů je vytvořen, který se zaměřuje na vzhled lokality. Nejprve se vytvoří ukázkového webu s soubor CSS pro ukázkové stránky. První skupiny stylů vytvořené jsou zaměřeno na poskytování kontejnery pro obsah v celém webu. V dalším kroku se přidají sadu stylů, které se zaměřují na zobrazení "stavu" prvky uživatelského rozhraní. Nakonec se vytvoří sadu styly pro kódování interakci uživatele. Jednotlivé styly se používá s prvků HTML na stránce ukázka. Je také diskuzi o použití ve stejný přístup implementace pro styly tento element rozložení ovládacích prvků na webu.
