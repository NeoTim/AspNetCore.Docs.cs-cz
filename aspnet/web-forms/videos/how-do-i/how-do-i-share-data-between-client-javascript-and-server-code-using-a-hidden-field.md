<a name="how-do-i-share-data-between-client-javascript-and-server-code-using-a-hidden-field"></a>[Jak na:] Sdílet Data mezi JavaScript klienta a kód serveru pomocí skryté pole
====================
podle [PEL Jan](https://twitter.com/chrispels)

V této video PEL Jan zobrazí jak sdílet data mezi kódu na straně serveru a používání jazyka JavaScript pomocí skryté pole na stránce technologie ASP.NET. V dalším kroku zjistěte, jak lze pomocí metody RegisterHiddenField v kódu na straně serveru prostřednictvím kódu programu vytvořit skryté pole na straně klienta. Kromě toho naleznete v části vytváření ovládacího prvku HiddenField server pro sdílení dat na stránku ASP.NET prostřednictvím kódu programu. Nakonec použijte metodu RegisterArrayDeclaration vytvořit kontejner v serverovém kódu pro ukládání více hodnot, který je vykreslen na stránku, kde byla přístupná pomocí jazyka JavaScript.
