<a name="how-do-i-configure-an-aspnet-web-site-for-a-net-framework-version"></a>[Jak na:] Konfigurace webu technologie ASP.NET pro verzi rozhraní .NET Framework
====================
podle [PEL Jan](https://twitter.com/chrispels)

V této video PEL Jan zobrazí jak nakonfigurovat webovou stránku ASP.NET použít konkrétní verzi ASP.NET/.NET Framework. Nejprve zjistit, jak určit, jaká verze rozhraní .NET Framework webu přidružen zobrazením mapu skriptů se v Internet Information Server (IIS). V tématu Jak mapu skriptů souvisí s kořenu ve službě IIS a pak se jednotlivých webů. Další informace týkající se nástroje ASP.NET IIS Registration (aspnet\_regiis.exe) slouží ke konfiguraci služby IIS nebo webovou stránku ASP.NET, které chcete použít konkrétní verzi rozhraní .NET Framework. Pomocí této informace najdete v tématu jak jeden web konfigurován pro použití jinou verzi rozhraní .NET Framework, než je výchozí nastavení pro službu IIS. Nakonec zkontrolujte dvě různé weby v rámci stejné služby IIS, kde jeden běží v rozhraní .NET Framework 2.0 a další části 1.1.
