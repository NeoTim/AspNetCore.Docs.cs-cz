V této video PEL Jan ukazuje, jak použít sledování SqlWebEventProvider stavu ASP.NET k protokolování chyb v databázi systému SQL Server webu. Potom informace o konfiguraci stavu monitorování v souboru web.config k zaznamenání události na webu technologie ASP.NET do nově vytvořené databáze systému SQL Server. V rámci této konfigurace najdete v části Jak v kořenovém souboru web.config v rozhraní .NET Framework 2.0 má poskytovatelů i definované události, které můžete využít při konfiguraci, sledování stavu. Tyto základní informace jsou základní němž lze vytvořit vlastní událostí, který protokolu konkrétní informace do databáze systému SQL Server na webu technologie ASP.NET.
