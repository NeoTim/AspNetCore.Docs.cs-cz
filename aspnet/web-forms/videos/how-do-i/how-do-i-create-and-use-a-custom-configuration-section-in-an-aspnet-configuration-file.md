<a name="how-do-i-create-and-use-a-custom-configuration-section-in-an-aspnet-configuration-file"></a>[Jak na:] Vytvořit a používat vlastní konfigurační oddíl v konfiguračním souboru ASP.NET
====================
podle [PEL Jan](https://twitter.com/chrispels)

V tomto videu se Jan PEL ukazují, jak rozšířit standardní sadu nastavení konfigurace ASP.NET s vlastní elementy XML konfigurace. Nejdříve si projděte postup vytvoření obslužnou rutinu oddílu konfigurace, která je vyžadována jak interpretovat a zpracovat informace ve vlastní elementy XML konfigurace. Během této zjistěte, jak použít deklarativní model pro definování chování obslužnou rutinu oddílu konfigurace. V dalším kroku najdete v části jak přidat vlastní oddíl obslužné rutiny a vlastní elementy do konfiguračního souboru technologie ASP.NET. Nakonec zkontrolujte, jak programově přistupovat k datům vlastní konfigurace.
