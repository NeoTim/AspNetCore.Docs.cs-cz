<a name="how-do-i-add-javascript-to-an-aspnet-page"></a>[Jak na:] Přidat JavaScript na stránku ASP.NET
====================
podle [PEL Jan](https://twitter.com/chrispels)

V tomto videu se zobrazí Jan PEL postup zahrnutí JavaScript na stránku ASP.NET pro použití na stránce klienta zajistit bohatší a robustnější uživatelské prostředí. Nejdříve si projděte postupy zahrnují JavaScript přímo v rámci stránky ASP.NET a odkaz skript v prvky na stránce HTML. V dalším kroku naleznete v části abstraktní JavaScript na externí soubor JavaScript, který nabízí výhodu v podobě ukládat do mezipaměti, prohlížeče a mohou být využívána více stránek. A konečně zjistěte, jak přidat JavaScript na stránku dynamicky za běhu, který má výhodu, včetně skriptů pro konkrétní situaci nebo kde není znám informace, jako je ID ovládacího prvku, dokud modulu runtime. Jako součást dynamické skript ukázka zjistit, jak přidat JavaScript, který lze automaticky spustit při načtení stránky a klikněte na události pro ovládací prvky tlačítka (Button, LinkButton a ImageButton).
