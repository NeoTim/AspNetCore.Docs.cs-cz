<a name="how-do-i-export-data-to-a-comma-delimited-csv-file-for-an-application-like-excel"></a>[Jak na:] Exportovat Data do textového souboru s oddělovači (CSV) pro aplikaci, například Excel
====================
podle [PEL Jan](https://twitter.com/chrispels)

V tomto videu Jan PEL ukazuje, jak provést data z databáze nebo jiného zdroje a exportovat je do souboru s oddělovači čárkami, který lze použít v aplikaci jako aplikace Excel. Sada dat je nejprve vytvořit jako objekt DataTable. V dalším kroku odpovědi pro aktuální požadavek na webovou stránku není zaškrtnuté a hlavičky a typu obsahu jsou nakonfigurované jako soubor csv. Skutečná data se pak přidá do datového proudu odpovědi první napsáním záhlaví sloupců pro soubor csv, za nímž následuje datových hodnot. Tento přístup může být užitečné, když uživatelé vyžadují o export dat, takže smí uživatel manipulovat místně v programu, například Excel.
