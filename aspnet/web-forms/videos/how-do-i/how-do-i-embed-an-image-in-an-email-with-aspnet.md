<a name="how-do-i-embed-an-image-in-an-email-with-aspnet"></a>[Jak na:] Vložit obrázek e-mailem s technologií ASP.NET
====================
podle [PEL Jan](https://twitter.com/chrispels)

Jan PEL ukazuje, jak pro vložení bitovou kopii do e-mailu s technologií ASP.NET. Zadá vytvoří webového formuláře (s pole pro, od, předmět a text), třídu AlternateView použije k vytvoření textu a HTML verzích e-mailu, ukládá bitovou kopii v instanci třídy LinkedResource, vloží do HTML AlternateView. Potom přidá obě verze objektu poštovní zpráva a odešle e-mail dvakrát, nejprve s přijetím HTML funkcích, které přináší a pak jako prostého textu. Obě e-mailové zprávy se zobrazují v aplikaci Outlook. Zobrazuje verzi jazyka HTML vložený obrázek; text verze neexistuje.
