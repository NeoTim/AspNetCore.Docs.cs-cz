<a name="how-do-i-use-the-aspnet-cache-object-to-cache-application-information"></a>[Jak na to]: použít objekt mezipaměti technologie ASP.NET pro ukládání do mezipaměti informace o aplikaci
====================
podle [PEL Jan](https://twitter.com/chrispels)

V této video PEL Jan ukazuje, jak použít objekt mezipaměti technologie ASP.NET pro ukládání informací o aplikaci. Ukázkové stránky se vytvoří jednoduchý uživatelského rozhraní a přidání kódu, který naplní mřížky s daty ze souboru XML disku. Pak je změněn kód pro ukládání do mezipaměti data v mezipaměti objekt aplikace poté, co je nejdřív načíst a načten z mezipaměti pro následné požadavky. Nakonec metody jsou přidány, aby použití absolutní a klouzavé čas na základě závislostí pro objekt uložená v mezipaměti.
