<a name="enabling-full-text-search-in-your-text-data"></a>Povolení fulltextového vyhledávání v textových datech
====================
podle [Microsoft](https://github.com/microsoft)

Fulltextové vyhledávání umožňuje ukládat velké části textu – dokonce i dokumenty Microsoft Wordu nebo jiných formátů souborů – do databáze a provádět složité dotazy na základě této textové dat ale byla očekávána skvělý výkon. Tato lekce ukazuje, jak nastavit fulltextové vyhledávání, věnujte zvláštní pozornost drobné rozdíly v jeho nastavení v systému SQL Server Express. Pak budeme postupovat vytváření fulltextových indexů, konfigurace nastavení naplnění indexu a provádění dotazů, které ukazují sílu predikátu CONTAINS a jeho mnoho, mnoho variant.
