<a name="how-do-i-return-json-formatted-data-for-an-ajax-call-in-an-aspnet-mvc-web-application"></a>Jak I: vrátit JSON naformátovaný Data ve webové aplikaci ASP.NET MVC volání AJAX?
====================
podle [PEL Jan](https://twitter.com/chrispels)

V tomto videu, které Jan PEL ukazuje, jak použít třídy JsonResult a Json, který vrátí instance tříd jako JSON ve formátu data. Nejprve se vytvoří ukázkové webové aplikaci MVC a kontroler a přidružené zobrazení jsou přidány informace o odběrateli. Pak třídu jednoduché zákazníka je vytvořen v rámci datového modelu. Funkce GetCustomer() se pak přidá do řadiče zákazníka k načtení dat pro jednoho zákazníka ve formátu JSON. Nakonec se skript přidá do zobrazení zákazníka, které používá jQuery volání jazyka AJAX do GetCustomer() k načtení dat zákazníka ve formátu JSON
