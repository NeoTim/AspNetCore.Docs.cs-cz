<a name="how-do-i-create-a-custom-html-helper-for-an-mvc-application"></a>Jak vytvořit I: Pomocník vlastní HTML pro aplikaci MVC?
====================
podle [PEL Jan](https://twitter.com/chrispels)

V tomto videu Jan PEL ukazuje, jak vytvořit vlastní HtmlHelper, který není k dispozici v standardní sady v aplikaci MVC. Ukázková aplikace MVC se nejprve vytvoří s ukázku řadiče a zobrazení otestovat vlastní HtmlHelper. V dalším kroku se vytvoří modul s veřejnou funkci, která je metody rozšíření, která představuje implementaci vlastní HtmlHelper. Je vlastního pomocného objektu pro vytváření &amp;lt; img&amp;gt; značky na stránce a přijímá několik příchozích parametry, včetně id, adresy url a alternativní text pro obrázek značky. Logika se pak přidá funkce pro vrácení dokončené &amp;lt; img&amp;gt; značky se zadanými informacemi. Vlastní HtmlHelper je použita na stránce ukázku zobrazíte bitovou kopii. Nakonec je vlastní HtmlHelper rozšířit, aby obsahovaly více konstruktor přepsání, které poskytují flexibilitu pro více snadno vytváření jiný &amp;lt; img&amp;gt; značky.
