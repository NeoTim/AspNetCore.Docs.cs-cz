<a name="jquery-cycle-299"></a>jQuery 2.99 cyklu
====================
Následující soubory jQuery cyklu jsou hostované na této sítě CDN:

- [https://ajax.aspnetcdn.com/ajax/jquery.cycle/2.99/jquery.cycle.all.js](https://ajax.aspnetcdn.com/ajax/jquery.cycle/2.99/jquery.cycle.all.js) &mdash; Plnou verzi modulu plug-in jQuery cyklu. Zahrnuje více než dvacet speciální efekty a všechny možnosti.
- [https://ajax.aspnetcdn.com/ajax/jquery.cycle/2.99/jquery.cycle.all.min.js](https://ajax.aspnetcdn.com/ajax/jquery.cycle/2.99/jquery.cycle.all.min.js) &mdash; Minifikovaný verze souboru výše. Pomocí této verze aplikacích v produkčním prostředí pro zlepšení výkonu vašeho webu.
- [https://ajax.aspnetcdn.com/ajax/jquery.cycle/2.99/jquery.cycle.js](https://ajax.aspnetcdn.com/ajax/jquery.cycle/2.99/jquery.cycle.js)  &mdash; Tato verze modulu plug-in jQuery cyklus je menší a obsahuje pouze efekt přechodu "fade".
- [https://ajax.aspnetcdn.com/ajax/jquery.cycle/2.99/jquery.cycle.min.js](https://ajax.aspnetcdn.com/ajax/jquery.cycle/2.99/jquery.cycle.min.js) &mdash; Minifikovaný verze souboru výše. Pomocí této verze aplikacích v produkčním prostředí pro zlepšení výkonu vašeho webu.
- [https://ajax.aspnetcdn.com/ajax/jquery.cycle/2.99/jquery.cycle.lite.js](https://ajax.aspnetcdn.com/ajax/jquery.cycle/2.99/jquery.cycle.lite.js) &mdash; I menší verzi modulu plug-in jQuery cyklu, obsahující pouze "/ roztmívací" přechod bez všechny možnosti.
- [https://ajax.aspnetcdn.com/ajax/jquery.cycle/2.99/jquery.cycle.lite.min.js](https://ajax.aspnetcdn.com/ajax/jquery.cycle/2.99/jquery.cycle.lite.min.js) &mdash; Minifikovaný verze souboru výše. Pomocí této verze aplikacích v produkčním prostředí pro zlepšení výkonu vašeho webu.
