<a name="jquery-cycle-288"></a>jQuery 2,88 cyklu
====================
Následující soubory cyklus jQuery jsou hostované na této CDN:

- http://AJAX.aspnetcdn.com/AJAX/jQuery.Cycle/2.88/jQuery.Cycle.all.js &mdash; plnou verzi modulu plug-in jQuery cyklu. Zahrnuje všechny více než dvacet zvláštní efekty a všechny možnosti.
- http://AJAX.aspnetcdn.com/AJAX/jQuery.Cycle/2.88/jQuery.Cycle.all.min.js &mdash; minifikovaný verzi souboru výše. Pomocí této verze pro výrobní aplikace lze zlepšit výkon webu.
- http://AJAX.aspnetcdn.com/AJAX/jQuery.Cycle/2.88/jQuery.Cycle.js &mdash; tuto verzi modulu plug-in jQuery cyklus je menší a obsahuje pouze efekt přechodu "Objevování".
- http://AJAX.aspnetcdn.com/AJAX/jQuery.Cycle/2.88/jQuery.Cycle.min.js &mdash; minifikovaný verzi souboru výše. Pomocí této verze pro výrobní aplikace lze zlepšit výkon webu.
- http://AJAX.aspnetcdn.com/AJAX/jQuery.Cycle/2.88/jQuery.Cycle.Lite.js &mdash; i menší verzi modulu plug-in jQuery cyklus, který obsahuje pouze přechodu "Objevování" bez všechny možnosti.
- http://AJAX.aspnetcdn.com/AJAX/jQuery.Cycle/2.88/jQuery.Cycle.Lite.min.js &mdash; minifikovaný verzi souboru výše. Pomocí této verze pro výrobní aplikace lze zlepšit výkon webu.
