# <a name="community-oss-authentication-options"></a>Možnosti ověřování komunity operačních systémů

Tato stránka obsahuje možnosti ověřování zadaný komunity open source pro ASP.NET Core. Tato stránka bude pravidelně aktualizovat, protože nové zprostředkovatele, které jsou k dispozici.

# <a name="oss-authentication-providers"></a>Zprostředkovatelé ověřování operačních systémů

Tento seznam řadí abecedně.

| Název | Popis |
|:--------------|:------------------|
| [AspNet.Security.OpenIdConnect.Server (ASOS)](https://github.com/aspnet-contrib/AspNet.Security.OpenIdConnect.Server) | Minimum-úrovně/protocol první OpenID Connect serveru rozhraní pro ASP.NET Core a OWIN nebo Katana |
| [IdentityServer4](https://identityserver.io/) | Architektura OpenID Connect a OAuth 2.0 pro ASP.NET Core - certifikované OpenID Foundation a v části zásady správného řízení Foundation rozhraní .NET |
| [OpenIddict](https://github.com/openiddict/openiddict-core) | Snadné použití serveru ASP.NET Core OpenID Connect  |
| [Cierge](https://github.com/pwdless/Cierge) | Ověřování OpenID Connect passwordless, drop-in   |

K získání svého poskytovatele tady přidat [upravit tuto stránku](https://github.com/login?return_to=https%3A%2F%2Fgithub.com%2Faspnet%2FDocs%2Fedit%2Fmaster%2Faspnetcore%2Fsecurity%2Fauthentication%2Fcommunity.md).
