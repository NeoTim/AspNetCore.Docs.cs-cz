## <a name="update-net-core-sdk-version-in-globaljson"></a>Aktualizace verze sady .NET Core SDK v souboru global.json

Pokud při cílení na konkrétní verzi sady .NET Core SDK spoléháte `version` na soubor [global.json,](/dotnet/core/tools/global-json) aktualizujte vlastnost na nainstalovanou verzi sady .NET 5.0 SDK. Příklad:

```diff
{
  "sdk": {
-    "version": "3.1.200"
+    "version": "5.0.100-preview.2.20176.6"
  }
}
```

## <a name="update-the-target-framework"></a>Aktualizace cílového rámce

V souboru projektu aktualizujte zástupný [název cílového rozhraní (TFM)](/dotnet/standard/frameworks) na `netcoreapp5.0`:

```diff
<Project Sdk="Microsoft.NET.Sdk.Web">

  <PropertyGroup>
-    <TargetFramework>netcoreapp3.1</TargetFramework>
+    <TargetFramework>netcoreapp5.0</TargetFramework>
  </PropertyGroup>

</Project>
```

## <a name="update-package-references"></a>Aktualizovat odkazy na balíčky

V souboru projektu `Microsoft.AspNetCore.*` aktualizujte `Microsoft.Extensions.*` `Version` atribut každého a odkazu na balíček na 5.0.0 nebo novější. Příklad:

```diff
<ItemGroup>
-    <PackageReference Include="Microsoft.AspNetCore.Mvc.NewtonsoftJson" Version="3.1.2" />
-    <PackageReference Include="Microsoft.Extensions.Caching.Abstractions" Version="3.1.2" />
+    <PackageReference Include="Microsoft.AspNetCore.Mvc.NewtonsoftJson" Version="5.0.0-preview.2.20167.3" />
+    <PackageReference Include="Microsoft.Extensions.Caching.Abstractions" Version="5.0.0-preview.2.20160.3" />
</ItemGroup>
```

## <a name="update-docker-images"></a>Aktualizace imitace Dockeru

Pro aplikace používající Docker použijte základní bitovou kopii, která obsahuje ASP.NET Core 5.0. Příklad:

```bash
docker pull mcr.microsoft.com/dotnet/core/aspnet:5.0
```
