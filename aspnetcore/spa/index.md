# <a name="use-the-single-page-application-templates-preview"></a>Použití šablon jednostránkové aplikace (preview)

> [!NOTE]
> Vydaná .NET Core 2.0.x SDK obsahuje šablony projektů pro úhlová reagují a s Redux reagovat. **Tato dokumentace není o tyto šablony vydaných projektu.** Tato dokumentace je pro další verze úhlová, reagují a reagují s – obnovení šablony, které Věříme, že pro odeslání v časná 2018.

## <a name="prerequisites"></a>Požadavky

* [.NET core SDK](https://www.microsoft.com/net/download), verze 2.0.0 nebo novější
* [Node.js](https://nodejs.org), verze 6 nebo novější

## <a name="installation"></a>Instalace

Spusťte následující příkaz k instalaci **preview** vydání šablony ASP.NET Core pro úhlová reagují a reagují s Redux:

```console
dotnet new --install Microsoft.DotNet.Web.Spa.ProjectTemplates::2.0.0-preview1-final
```
