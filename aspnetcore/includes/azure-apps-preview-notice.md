> [!IMPORTANT]
> **ASP.NET Core 3,0 s Azure App Service**
>
> ASP.NET Core 3,0 ještě není nasazený do Azure App Service. Doufáme, že Azure App Service Brzo poskytneme ASP.NET Core 3. Hostování aplikace, která používá ASP.NET Core 3,0:

* Považovat ASP.NET Core 3,0 jako verzi Preview pro Azure App Service nasazení.
* Další informace najdete v tématu [nasazení ASP.NET Core verze Preview do Azure App Service](xref:host-and-deploy/azure-apps/index#deploy-aspnet-core-preview-release-to-azure-app-service).
