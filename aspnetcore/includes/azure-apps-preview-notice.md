> [!IMPORTANT]
> **Verze Preview ASP.NET Core s Azure App Service**
>
> Verze Preview ASP.NET Core nejsou ve výchozím nastavení nasazené do Azure App Service. Pokud chcete hostovat aplikaci, která používá verzi Preview služby ASP.NET Core, přečtěte si téma [nasazení ASP.NET Core verze Preview do Azure App Service](xref:host-and-deploy/azure-apps/index#deploy-aspnet-core-preview-release-to-azure-app-service).
