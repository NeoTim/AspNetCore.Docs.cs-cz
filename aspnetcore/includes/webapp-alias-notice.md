> [!NOTE]
> Pokud `dotnet new webapp <OPTIONS>` příkaz zatížení [dotnet nové](/dotnet/core/tools/dotnet-new) příkaz. nápověda místo vytvoření nové aplikace stránky Razor, nainstalujte [.NET Core SDK 2.1.300 nebo novější](https://www.microsoft.com/net/download/archives). Od verze rozhraní .NET Core SDK 2.1.300 `webapp` *krátký název* je alias pro `razor`.
