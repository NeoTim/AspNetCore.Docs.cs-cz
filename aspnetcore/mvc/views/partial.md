## <a name="partial-view-discovery"></a><span data-ttu-id="7698b-161">Částečné zobrazení zjišťování</span><span class="sxs-lookup"><span data-stu-id="7698b-161">Partial view discovery</span></span>

```cshtml
// Uses a view in current folder with this name.
// If none is found, searches the Shared folder.
<partial name="_ViewName" />

// A view with this name must be in the same folder
<partial name="_ViewName.cshtml" />

// Locate the view based on the app root.
// Paths that start with "/" or "~/" refer to the app root.
<partial name="~/Views/Folder/_ViewName.cshtml" />
<partial name="/Views/Folder/_ViewName.cshtml" />

// Locate the view using a relative path
<partial name="../Account/_LoginPartial.cshtml" />
```

<span data-ttu-id="7698b-164">V předchozím příkladu částečné pomocné rutiny značky, který vyžaduje ASP.NET Core 2.1 nebo novější.</span><span class="sxs-lookup"><span data-stu-id="7698b-164">The preceding example uses the Partial Tag Helper, which requires ASP.NET Core 2.1 or later.</span></span> <span data-ttu-id="7698b-165">Následující příklad používá asynchronní pomocných rutin HTML stejný úkol provést.</span><span class="sxs-lookup"><span data-stu-id="7698b-165">The following example uses asynchronous HTML Helpers to accomplish the same task.</span></span>

::: moniker-end

```cshtml
// Uses a view in current folder with this name.
// If none is found, searches the Shared folder.
@await Html.PartialAsync("_ViewName")

// A view with this name must be in the same folder
@await Html.PartialAsync("_ViewName.cshtml")

// Locate the view based on the app root.
// Paths that start with "/" or "~/" refer to the app root.
@await Html.PartialAsync("~/Views/Folder/_ViewName.cshtml")
@await Html.PartialAsync("/Views/Folder/_ViewName.cshtml")

// Locate the view using a relative path
```
